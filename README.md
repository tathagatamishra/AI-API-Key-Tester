
  # AI API Key Tester

- A very simple app which will help user to test their ai api keys.
- There is a option to choose ai tool such openai, claude, gemini, github copilot. For now only these 4 tools.
- User will select a tool out of these 4. then user will enter their api key.
- Then select model.
- Enter simple prompt, and get simple output.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
